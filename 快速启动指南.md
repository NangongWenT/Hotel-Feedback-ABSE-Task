## ⚠️ 关于 GPU 警告

启动时可能会看到以下警告：
```
NVIDIA GeForce RTX 5060 Laptop GPU with CUDA capability sm_120 is not compatible...
```

**这是正常的！** PyTorch 会自动使用兼容的计算能力，GPU 加速仍然有效。你可以通过以下方式验证：

```powershell
.\venv\Scripts\python.exe -c "import torch; print('CUDA:', torch.cuda.is_available()); print('GPU:', torch.cuda.get_device_name(0))"
```

如果显示 `CUDA: True`，说明 GPU 正常工作。
